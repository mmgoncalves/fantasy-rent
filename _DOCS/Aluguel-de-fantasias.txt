/* INTRODUÇÃO */
Aplicativo para controle de aluguel de fantasias.
O cliente precisa cadastrar todas as fantasias, a quantidade de cada,
a cor, o tamanho, o sexo.

Para alugar uma fantasia, é necessario o cpf e o telefone do usuario, a data do aluguel e a data da entrega, e a quantidade.

Será necessário um formulário para o cadastro das fantasías.
Outro formulário para inserir um novo aluguel.
E uma página que lista as fantasias alugadas, onde é possível mudar o status para ENTREGUE.
Uma página de relatórios e estatísticas.

/* TECNOLOGIAS */
-Ionic
-Bower
-Gulp
-Sass
-Banco SqLite
